## Overview

Authentication routes handle user login, logout, and password recovery. These routes are publicly accessible and do not require authentication middleware.

## Middleware Reference

<Warning>
  All protected routes require the `PreventBackHistoryMiddleware` and `auth` middleware. This prevents users from accessing pages via browser back button after logout.
</Warning>

### PreventBackHistoryMiddleware

Prevents browser caching by setting appropriate headers:
- `Cache-Control: no-store, no-cache, must-revalidate, max-age=0`
- `Pragma: no-cache`
- `Expires: Fri, 01 Jan 1990 00:00:00 GMT`

**Source**: PreventBackHistoryMiddleware.php:16-26
