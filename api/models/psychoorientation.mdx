## Overview

The `Psychoorientation` model represents psychological orientation reports created by school psychologists (psicoorientadores) for students requiring specialized support. It documents evaluations, interventions, and recommendations.

**Namespace:** `App\Models\Psychoorientation`

**Extends:** `Illuminate\Database\Eloquent\Model`

**Traits:**
- `HasFactory` - Enables model factories for testing

## Model Definition

```php
class Psychoorientation extends Model
{
    use HasFactory;
}
```

## Fillable Fields

These fields are permitted for mass assignment when creating or updating psycho-orientation records.

<ParamField body="psychologist_writes" type="integer" required>
  Foreign key to Users_teacher model - the psychologist who wrote the report
</ParamField>

<ParamField body="id_user_student" type="integer" required>
  Foreign key to Users_student model - the student being evaluated
</ParamField>

<ParamField body="age_student" type="integer">
  Student's age at the time of evaluation
</ParamField>

<ParamField body="group_student" type="string">
  Student's class group (e.g., "6A", "7B")
</ParamField>

<ParamField body="director_group_student" type="string">
  Name of the student's group director/homeroom teacher
</ParamField>

<ParamField body="title_report" type="string" required>
  Title or subject of the psychological report
</ParamField>

<ParamField body="reason_inquiry" type="text" required>
  Reason for the psychological inquiry or evaluation
</ParamField>

<ParamField body="recomendations" type="text">
  Professional recommendations and suggested interventions
</ParamField>

<ParamField body="annex_one" type="string">
  File path or reference to an attached document (evaluations, test results, etc.)
  
  **Note:** This is a critical field for document attachments
</ParamField>

## Relationships

### BelongsTo Relationships

<Expandable title="user_psychology()">
  Returns the psychologist/psycho-orientator who wrote the report.

  ```php
  public function user_psychology()
  {
      return $this->belongsTo(Users_teacher::class, 'psychologist_writes');
  }
  ```

  **Foreign Key:** `psychologist_writes`

## Usage Examples

### Creating a Psycho-orientation Report

```php
use App\Models\Psychoorientation;
use App\Models\Users_student;

$student = Users_student::find(15);

$report = Psychoorientation::create([
    'psychologist_writes' => 5, // ID of the psychologist
    'id_user_student' => $student->id,
    'age_student' => $student->age,
    'group_student' => $student->group->name,
    'director_group_student' => $student->group->director->name ?? 'N/A',
    'title_report' => 'Evaluación de adaptación escolar',
    'reason_inquiry' => 'El estudiante presenta dificultades en la integración social con sus compañeros y muestra signos de ansiedad durante actividades grupales.',
    'recomendations' => 'Se recomienda: 1) Implementar actividades de integración gradual, 2) Seguimiento semanal por parte del psicoorientador, 3) Reunión con padres de familia para establecer estrategias de apoyo en casa.',
    'annex_one' => 'uploads/psycho/evaluation_2024_03_15.pdf',
]);
```

### Querying Psycho-orientation Reports

```php
// Get all reports with psychologist and student information
$reports = Psychoorientation::with([
    'user_psychology',
    'user_student.group',
    'user_student.degree'
])
->orderBy('created_at', 'desc')
->get();

foreach ($reports as $report) {
    echo "Title: {$report->title_report}";
    echo "Student: {$report->user_student->name}";
    echo "Psychologist: {$report->user_psychology->name}";
    echo "Date: {$report->created_at->format('Y-m-d')}";
    echo "---";
}
```

## Integration with Referrals

```php
// Get students with referrals and their psycho-orientation follow-up
$studentCases = Users_student::with([
    'referrals' => function ($query) {
        $query->orderBy('created_at', 'desc');
    },
    'psychoorientations' => function ($query) {
        $query->orderBy('created_at', 'desc');
    }
])
->has('referrals')
->get();

foreach ($studentCases as $student) {
    echo "Student: {$student->name}";
    echo "Referrals: {$student->referrals->count()}";
    echo "Psycho-orientations: {$student->psychoorientations->count()}";
    
    // Check if latest referral has been followed up
    $latestReferral = $student->referrals->first();
    $latestPsycho = $student->psychoorientations->first();
    
    if ($latestReferral && (!$latestPsycho || $latestReferral->created_at > $latestPsycho->created_at)) {
        echo "⚠ Needs psycho-orientation follow-up";
    }
}
```
