## Overview

The `Users_teacher` model represents teaching staff and administrators in the Bethlemitas Student Support System. It handles authentication, role-based permissions, academic assignments, and relationships with students and courses.

**Namespace:** `App\Models\Users_teacher`

**Extends:** `Illuminate\Foundation\Auth\User` (Authenticatable)

**Traits:**
- `HasFactory` - Enables model factories for testing
- `HasRoles` (Spatie) - Provides role-based permissions
- `Notifiable` - Enables notifications

## Model Definition

```php
class Users_teacher extends Authenticatable
{
    use HasFactory, HasRoles, Notifiable;

    protected $table = 'users_teachers';
    protected $guard_name = 'web';
}
```

<Expandable title="state()">
  Returns the current state/status of the teacher.

## Academic Relationships

### BelongsToMany Relationships

<Expandable title="areas()">
  Returns all academic areas/subjects assigned to this teacher.

  ```php
  public function areas()
  {
      return $this->belongsToMany(
          Area::class,
          'teachers_areas_groups',
          'id_teacher',
          'id_area'
      )->withPivot('id_group');
  }
  ```

  **Pivot Table:** `teachers_areas_groups`
  
  **Related Model:** `Area`
  
  **Pivot Data:** Includes `id_group` to show which group the teacher teaches this area to

  **Usage:**
  ```php
  $teacher = Users_teacher::find(1);
  $areas = $teacher->areas;
  
  foreach ($areas as $area) {
      echo $area->name; // "Mathematics", "Science", etc.
      echo "Group: " . $area->pivot->id_group;
  }
  ```
</Expandable>

<Expandable title="scopeDocentesPsicoCoordinador()">
  Filters to teachers with specific roles: docente, psicoorientador, or coordinador.

  ```php
  public function scopeDocentesPsicoCoordinador($query)
  {
      return $query->whereHas('roles', function ($q) {
          $q->whereIn('name', [
              'docente',
              'psicoorientador',
              'coordinador'
          ]);
      });
  }
  ```

  **Usage:**
  ```php
  $staff = Users_teacher::docentesPsicoCoordinador()->get();
  ```
</Expandable>

## Usage Examples

### Creating a Teacher

```php
use App\Models\Users_teacher;
use Illuminate\Support\Facades\Hash;

$teacher = Users_teacher::create([
    'name' => 'María',
    'lastname' => 'García',
    'email' => 'maria.garcia@bethlemitas.edu.co',
    'password' => Hash::make('secure-password'),
    'document' => '98765432',
    'phone' => '3009876543',
    'id_state' => 1, // Active
    'group_director' => null,
]);

// Assign role
$teacher->assignRole('docente');
```

### Querying with Scopes

```php
// Get all active teachers who are psycho-orientators
$psychologists = Users_teacher::activos()
    ->whereHas('roles', function ($q) {
        $q->where('name', 'psicoorientador');
    })
    ->get();

// Get all active teaching staff
$staff = Users_teacher::activos()
    ->docentesPsicoCoordinador()
    ->get();
```

### Working with Relationships

```php
// Get teacher with all academic assignments
$teacher = Users_teacher::with([
    'areas',
    'groups',
    'director',
    'state'
])->find(1);

// Get teaching schedule
foreach ($teacher->areasGroups as $assignment) {
    $area = Area::find($assignment->id_area);
    $group = Group::find($assignment->id_group);
    echo "{$teacher->name} teaches {$area->name} to group {$group->name}";
}

// Check if teacher is a group director
if ($teacher->group_director) {
    echo "{$teacher->name} directs group: {$teacher->director->name}";
}
```

### Authentication

```php
// Login
use Illuminate\Support\Facades\Auth;

$credentials = [
    'email' => 'maria.garcia@bethlemitas.edu.co',
    'password' => 'password123'
];

if (Auth::guard('web')->attempt($credentials)) {
    $teacher = Auth::user();
    echo "Welcome, {$teacher->name}!";
}
```

### Role-Based Queries

```php
// Get all coordinators
$coordinators = Users_teacher::role('coordinador')->get();

// Check if teacher has specific role
$teacher = Users_teacher::find(1);
if ($teacher->hasRole('psicoorientador')) {
    // Access psycho-orientation features
    $degrees = $teacher->degrees;
}
```
