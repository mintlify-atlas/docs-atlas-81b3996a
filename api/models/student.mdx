## Overview

The `Users_student` model represents students in the Bethlemitas Student Support System. It manages student profiles, academic assignments, and relationships with teachers, referrals, and psycho-orientation records.

**Namespace:** `App\Models\Users_student`

**Extends:** `Illuminate\Database\Eloquent\Model`

**Traits:**
- `HasFactory` - Enables model factories for testing
- `HasRoles` (Spatie) - Provides role-based permissions

## Model Definition

```php
class Users_student extends Model
{
    use HasFactory, HasRoles;

    protected $guard_name = 'web';
}
```

### HasOne Relationships

<Expandable title="latestReferral()">
  Returns the most recent referral record for this student.

  ```php
  public function latestReferral()
  {
      return $this->hasOne(Referral::class, 'id_user_student')->latestOfMany();
  }
  ```

  **Foreign Key:** `id_user_student`
  
  **Related Model:** [Referral](/api/models/referral)

  **Usage:**
  ```php
  $student = Users_student::find(1);
  $latest = $student->latestReferral;
  
  if ($latest) {
      echo "Latest referral: " . $latest->reason;
  }
  ```

## Usage Examples

### Creating a Student

```php
use App\Models\Users_student;

$student = Users_student::create([
    'number_documment' => '1234567890',
    'name' => 'Juan',
    'last_name' => 'Pérez',
    'age' => 12,
    'birthplace' => 'Medellín',
    'address' => 'Calle 123 #45-67',
    'type_documment' => 'TI',
    'phone' => '3001234567',
    'email' => 'juan.perez@example.com',
    'victim_conflict' => false,
    'id_degree' => 6,
    'id_group' => 2,
    'id_institution' => 1,
    'sent_by' => 5,
    'id_state' => 1,
    'activation_period' => '2024-1',
]);
```

### Querying with Relationships

```php
// Get student with all relationships
$student = Users_student::with([
    'teacher',
    'degree',
    'group',
    'referrals',
    'psychoorientations',
    'latestReferral'
])->find(1);

// Access related data
echo "Student: {$student->name} {$student->last_name}";
echo "Grade: {$student->degree->name}";
echo "Group: {$student->group->name}";
echo "Teacher: {$student->teacher->name}";
echo "Total Referrals: " . $student->referrals->count();
```

### Filtering Students

```php
// Active students in a specific grade
$students = Users_student::where('id_degree', 6)
    ->where('id_state', 1)
    ->get();

// Students with referrals
$studentsWithReferrals = Users_student::has('referrals')
    ->with('latestReferral')
    ->get();

// Conflict victims
$victimStudents = Users_student::where('victim_conflict', true)
    ->get();
```
