## Overview

The PIAR and DUA Management module provides specialized educational support for students with diverse learning needs. PIAR (Plan Individual de Ajustes Razonables) creates individualized adjustment plans, while DUA (Diseño Universal para el Aprendizaje) implements universal learning design principles.

## Understanding PIAR and DUA

## Teacher Access to PIAR/DUA Students

Teachers can view students in PIAR or DUA status for their assigned groups (`app/Http/Controllers/CreateReferralController.php:222`):

```php
public function addMinutes(Request $request)
{
    $id_teacher = Auth::id();
    
    // Get groups this teacher is assigned to
    $id_load_groups = Users_load_group::where('id_user_teacher', $id_teacher)
        ->pluck('id_group');
    
    // Get students in PIAR or DUA status from those groups
    $query = Users_student::whereHas('states', function ($q) {
        $q->whereIn('state', ['en PIAR', 'en DUA']);
    })->whereIn('id_group', $id_load_groups);
    
    // Apply search filter
    if ($request->filled('search')) {
        $searchTerm = $request->input('search');
        $query->where(function ($q) use ($searchTerm) {
            $q->where('name', 'LIKE', '%' . $searchTerm . '%')
                ->orWhere('last_name', 'LIKE', '%' . $searchTerm . '%')
                ->orWhere('number_documment', 'LIKE', '%' . $searchTerm . '%')
                ->orWhereRaw("CONCAT(name, ' ', last_name) LIKE ?", ['%' . $searchTerm . '%']);
        });
    }
    
    $students = $query->orderBy('name', 'asc')
        ->orderBy('last_name', 'asc')
        ->paginate(15);
    
    return view('teacher.addMinutes', compact('students'));
}
```

<Note>
  Teachers can only create PIAR/DUA documents for students in groups they teach. This ensures teachers work with students they know.
</Note>

## Viewing PIAR/DUA Students

### Psychologist View

Psychologists can filter students by state:

```php
// View students in PIAR
public function index_students_piar_psico(Request $request)
{
    return $this->studentsByState($request, 'en PIAR', 'En PIAR', 'psico.students.piar');
}

// View students in DUA
public function index_students_dua_psico(Request $request)
{
    return $this->studentsByState($request, 'en DUA', 'En DUA', 'psico.students.dua');
}

// Shared filter method
private function studentsByState(Request $request, string $state, string $label, string $routeName)
{
    $id_psico = Auth::id();
    
    $load_degree = Users_load_degree::where('id_user', $id_psico)
        ->pluck('id_degree')
        ->toArray();
    
    $query = Users_student::whereHas('states', function ($q) use ($state) {
            $q->where('state', $state);
        })
        ->whereIn('id_degree', $load_degree);
    
    // Check if students have psychological report attachments
    $students = $query
        ->withExists(['psychoorientations as has_annex' => function ($q) {
            $q->whereNotNull('annex_one');
        }])
        ->orderBy('name')
        ->orderBy('last_name')
        ->paginate(15);
    
    return view('psycho.listStudentsByState', [
        'students'   => $students,
        'stateLabel' => $label,
        'route'      => route($routeName),
    ]);
}
```

## Document History and Tracking

Both PIAR and DUA documents are:
- **Timestamped**: Automatic `created_at` and `updated_at` timestamps
- **Linked to periods**: Can track progress across academic periods
- **Teacher-attributed**: Always know who created each document
- **Student-specific**: Complete history per student

### Accessing Historical Documents

Through the student history view (`/student/history/{id}`), users can access:
- All PIAR documents created for a student
- All DUA activities designed for a student
- Chronological progression of support
- Evolution of adjustments over time

## Integration with Other Modules

### From Referral to PIAR

1. Teacher creates referral → Student is "activo"
2. Psychologist evaluates → Creates report
3. Psychologist changes state → Student is "en PIAR"
4. Teachers create PIAR documents → Student receives adjustments
5. Continuous monitoring → Update PIAR as needed

### DUA as Alternative or Complement

DUA can serve as:
- An alternative to PIAR for students needing less individualization
- A complement to PIAR for comprehensive support
- A preventative approach for all students

## Reporting and Analytics

The system enables tracking:
- Number of students in PIAR vs. DUA
